## 🔄 协作工作流程

```
产品需求 → @ProjectManager分析 → 分配专业代理 → 协作开发 → @TestAI验证 → 交付
```

### 标准流程示例
1. **需求分析**：产品经理描述功能需求
2. **任务分解**：ProjectManager分析并分配给对应代理
3. **专业设计**：各代理在专业领域深度设计
4. **协作开发**：多代理协作实现复杂功能
5. **质量保证**：@TestAI进行全面测试验证
6. **交付验收**：产品经理验收最终成果
