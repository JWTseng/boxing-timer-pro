🧪 TESTAI - 预设时间记忆功能验证

@UIAI + @CMAI + @TESTAI 协作：验证时间选择器智能记忆用户偏好

📋 功能说明

🎯 测试目标：验证时间选择器能智能记忆用户自定义时间

✨ 期望行为：
• 用户通过滚轮调整时间后，最接近的预设按钮会自动更新
• 系统会记住用户最常用的时间设置
• 下次打开时，预设按钮显示用户自定义的时间
• 智能替换策略：替换最接近的预设值（差异>5秒时生效）

🔬 测试场景

场景1：Prepare阶段预设更新

  • 点击Prepare时间（默认00:10）打开时间选择器
  • 注意预设按钮显示：00:10、00:20、01:00
  • 通过滚轮调整到00:15
  • 点击确认
  • 再次打开，预设按钮应更新为：00:10、00:15、01:00

场景2：持久化存储验证

  • 调整任意阶段时间并确认
  • 刷新页面（F5或Cmd+R）
  • 再次打开时间选择器
  • 验证预设按钮保持用户自定义值

场景3：智能替换策略

  • 设置时间为00:18（接近00:20）
  • 系统应替换00:20为00:18
  • 设置时间为00:21（差异小于5秒）
  • 系统不应更新预设（保持原值）
